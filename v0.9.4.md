# v0.9.4

_Signed-out sign-in now works._ When you try to create a meeting while signed
out of Yaw MCP, the "Sign in to Yaw" toast now opens the team sign-in panel
directly. One click from the error to the form.

The Mac arm64 build is unchanged this cycle; Mac users on `brew install vew`
stay on v0.9.3 until the next Mac-arm64 release ships from a Mac build host.

## Install

- Windows arm64 (auto-update from v0.9.x): already done by the in-app updater.
- Windows x64, Linux (.deb/.rpm/.zip): https://downloads.vew.sh
- Scoop: `scoop update vew`
- Mac (Homebrew): stays on v0.9.3 -- v0.9.4 Mac zip not produced this cycle.
