# vew Releases

Public release notes for [vew](https://vew.sh) — a browser, with an agent built in.

The vew app source is private; this repo is the public record of what ships in each version.

## Install

**Windows** (x64 & ARM64):

PowerShell:
```
irm https://vew.sh/install-win.ps1 | iex
```

Git Bash / WSL:
```
curl -fsSL https://vew.sh/install-win.sh | sh
```

**macOS**:
```
curl -fsSL https://vew.sh/install-mac.sh | sh
```

**Linux**:
```
curl -fsSL https://vew.sh/install-linux.sh | sh
```

Or grab a platform archive directly from <https://vew.sh>.
